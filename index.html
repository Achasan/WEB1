WEB은 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for
creating web pages and web applicatiions.

With Cascading Style Sheets (CSS) and JavaScript is
forms a thread of cornerstone technologies.

HTML elements are the building blocks of HTML pages.
With HTML constructs, images and other oibjects, such as
interactive forms, may be embedded into the rendered page.
It provides a means to create structured documents headings, paragraphs,
lists, links, quotes and other items.
HTML elements are delineated by tags, written using angle brackets.
